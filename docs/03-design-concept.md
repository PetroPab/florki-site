# 03. Дизайн-концепция

Визуальное направление сайта florki.ru.
Источник правды по эстетике, цвету, типографике,
композиции и микро-взаимодействиям.

---

## 1. Большая идея

### 1.1 Концепция: **Botanical Glassmorphism**

Метафора — **витрина оранжереи**. Сайт работает
как стеклянный сосуд: внутри — живая зелень
(фотографии работ), снаружи — мягкие стеклянные
слои, через которые проступает свет.

**Это не «модный glassmorphism из Dribbble».**
Это инструмент рассказать историю: каждый
флорариум — мир под стеклом, и сам сайт —
тоже стекло, через которое смотришь на этот мир.

### 1.2 Три кита

1. **Тёплая ботаника** — не «зелёненький с белым»,
   а сепия, мох, янтарь, кость.
2. **Гуманистический серив** — Fraunces вместо
   очередного Inter. Буквы как стебли:
   живые, чуть неровные, дышат.
3. **Стеклянные слои** — `backdrop-blur` +
   полупрозрачные плашки, имитирующие глубину
   витрины.

### 1.3 Чего точно НЕТ

- ❌ Фиолетово-синие градиенты Web3 / SaaS
- ❌ Минимализм-стерильность Apple-клона
- ❌ Bento-сетки как у всех в 2024
- ❌ «Stripe-style» иллюстрации в трёх цветах
- ❌ Брутализм с Arial Black
- ❌ Чёрный текст `#000` на белом `#fff`
- ❌ Generic gradient-mesh hero

## 3. Типографика

### 3.1 Шрифты

**Заголовки** — `Fraunces` (Google Fonts)

- Гуманистический серив с переменными осями
- Оси: `opsz` (12–144), `wght` (100–900), `SOFT`, `WONK`
- Подключаем варианты: 400, 500, 600, 700
- `WONK=1` для крупных заголовков (h1, h2) —
  «дикие» курсивные росчерки

**Основной текст** — `Inter` (Google Fonts)

- Гротеск с отличной читаемостью
- Веса: 400, 500, 600
- Используем «Optical Sizing» через `font-feature-settings`

> Альтернатива для основного текста — **Manrope**
> или **Geist**. Окончательно выбираем после
> прототипа Hero.

**Подключение:**

```ts
// app/fonts.ts
import { Fraunces, Inter } from 'next/font/google';

export const fraunces = Fraunces({
  subsets: ['latin', 'cyrillic'],
  variable: '--font-display',
  axes: ['opsz', 'SOFT', 'WONK'],
  display: 'swap',
});

export const inter = Inter({
  subsets: ['latin', 'cyrillic'],
  variable: '--font-body',
  display: 'swap',
});
```

### 3.2 Типографическая шкала

Все размеры — `clamp()` для плавного скейлинга
между mobile и desktop.

```css
--text-xs: clamp(0.75rem, 0.72rem + 0.15vw, 0.81rem);
--text-sm: clamp(0.875rem, 0.84rem + 0.18vw, 0.94rem);
--text-base: clamp(1rem, 0.96rem + 0.2vw, 1.06rem);
--text-lg: clamp(1.125rem, 1.07rem + 0.28vw, 1.19rem);
--text-xl: clamp(1.25rem, 1.15rem + 0.5vw, 1.5rem);
--text-2xl: clamp(1.5rem, 1.35rem + 0.75vw, 1.875rem);
--text-3xl: clamp(1.875rem, 1.6rem + 1.4vw, 2.5rem);
--text-4xl: clamp(2.25rem, 1.85rem + 2vw, 3.25rem);
--text-5xl: clamp(2.75rem, 2.1rem + 3.2vw, 4.5rem);
--text-6xl: clamp(3.5rem, 2.5rem + 5vw, 6rem);
```

### 3.3 Стили по ролям

| Роль        | Шрифт    | Размер  | Вес | Letter    | Line |
| ----------- | -------- | ------- | --- | --------- | ---- |
| H1 hero     | Fraunces | 5xl–6xl | 500 | -0.02em   | 1.05 |
| H1 страница | Fraunces | 4xl–5xl | 500 | -0.02em   | 1.1  |
| H2          | Fraunces | 3xl–4xl | 500 | -0.015em  | 1.15 |
| H3          | Fraunces | 2xl–3xl | 500 | -0.01em   | 1.2  |
| H4          | Fraunces | xl–2xl  | 500 | -0.005em  | 1.3  |
| Body L      | Inter    | lg      | 400 | -0.005em  | 1.6  |
| Body M      | Inter    | base    | 400 | 0         | 1.6  |
| Body S      | Inter    | sm      | 400 | 0         | 1.5  |
| Caption     | Inter    | xs      | 500 | 0.02em    | 1.4  |
| Метка/бейдж | Inter    | xs      | 600 | 0.06em UC | 1    |

**UC** — uppercase. Бейджи и метки — заглавными
с трекингом.

### 3.4 Тонкости

- **Висячие союзы и предлоги** — через `&nbsp;`
  в кириллице («в», «на», «по», «к», «о», «и», «а»)
- **Тире** — длинное (—), не дефис
- **Кавычки** — «ёлочки» снаружи, „лапки" внутри
- **Цифры с табулярным выравниванием** для цен:
  ```css
  font-feature-settings:
    'tnum' 1,
    'lnum' 1;
  ```
- **Лигатуры Fraunces** активны по умолчанию

## 5. Стекло (Glassmorphism) — фирменный приём

### 5.1 Базовые стеклянные слои

**Light Glass** (на светлом фоне)

```css
.glass-light {
  background: rgba(253, 251, 246, 0.65);
  backdrop-filter: blur(20px) saturate(120%);
  -webkit-backdrop-filter: blur(20px) saturate(120%);
  border: 1px solid rgba(107, 79, 42, 0.08);
  box-shadow:
    0 1px 0 0 rgba(255, 255, 255, 0.5) inset,
    0 8px 32px -8px rgba(61, 44, 23, 0.08);
}
```

**Frosted Glass** (на фото / тёмном)

```css
.glass-frosted {
  background: rgba(253, 251, 246, 0.4);
  backdrop-filter: blur(24px) saturate(140%);
  border: 1px solid rgba(255, 255, 255, 0.3);
  box-shadow:
    0 1px 0 0 rgba(255, 255, 255, 0.4) inset,
    0 12px 40px -8px rgba(0, 0, 0, 0.15);
}
```

**Tinted Glass** (с лёгким зелёным оттенком)

```css
.glass-tinted {
  background: rgba(232, 239, 216, 0.5);
  backdrop-filter: blur(20px);
  border: 1px solid rgba(107, 130, 69, 0.15);
}
```

### 5.2 Где применяем

- **Хедер** на скролле — glass-light
- **Карточка товара** — glass-light с hover
- **Hero CTA-блок** поверх фото — glass-frosted
- **Модалка заказа** — glass-light на блюре фона
- **Бейджи статуса** — solid (без стекла)

### 5.3 Где НЕ применяем

- ❌ Основной фон страницы (просто `cream-50`)
- ❌ Длинные текстовые блоки (читаемость падает)
- ❌ Футер (там обычный фон `cream-100`)
- ❌ Всё подряд (приём должен оставаться приёмом)

### 5.4 Fallback для старых браузеров

```css
@supports not (backdrop-filter: blur(20px)) {
  .glass-light {
    background: rgba(253, 251, 246, 0.95);
  }
}
```

## 9. Фотография

### 9.1 Принципы

- Только **реальные работы** студии
- Стиль: **естественный свет**, мягкие тени,
  тёплый баланс белого (5500–6000K)
- Фон: однотонный (бежевый / древесный /
  светло-серый), без хаоса
- Композиция: центральный объект, воздух вокруг
- Постобработка: минимальная, **без** инстаграм-фильтров

### 9.2 Запрещено

- ❌ Стоковые фото пальм / монстер
- ❌ Скриншоты из Pinterest
- ❌ Фото с водяными знаками
- ❌ Перенасыщенные «инстаграмные» цвета
- ❌ HDR-эффект с ореолами
- ❌ Зернистость / film grain ради «эстетики»

### 9.3 Технические требования

- Формат исходника: JPG / RAW
- На сайт: **WebP** (next/image конвертирует)
- Качество: 80–85 для WebP
- Размеры (long side):
  - Превью карточки: 800px
  - Главное фото товара: 1600px
  - Hero / galleries: до 2400px
- Alt-текст: обязательно, осмысленный

### 9.4 Соотношения сторон

- Карточка товара: **4:5** (вертикальная)
- Hero / cover: **16:9** или **21:9**
- Галерея masonry: разные, но не уже **3:4**
  и не шире **16:9**
- Аватары в отзывах: **1:1** (если есть)
- Обложки статей: **16:9**

## 10. Анимации и движение

### 10.1 Принципы

- **Цель важнее красоты.** Каждая анимация
  объясняет состояние или направляет внимание.
- **Скорость:** 150–300ms для UI, 400–600ms
  для входов секций.
- **Easing:** `cubic-bezier(0.22, 1, 0.36, 1)`
  (out-expo, мягкий выход)
- **Reduced motion:** все анимации отключаются
  или сокращаются до fade ≤ 100ms.

### 10.2 Где используем

- Hover карточки: `translateY(-4px)` + shadow
- Появление секций при скролле: `opacity 0 → 1`
  - `y: 20 → 0`, **один раз** (не на каждый скролл)
- Открытие модалки: fade + scale
- Lightbox: `layoutId` через Framer Motion
- Меню (мобильное): slide справа
- Кнопки CTA: лёгкий «pulse» glow на focus

### 10.3 Где НЕ используем

- ❌ Parallax на весь сайт
- ❌ Custom cursor
- ❌ Бесконечный autoplay карусели
- ❌ Печатающийся текст
- ❌ Сложные SVG-морфы без цели
- ❌ Stagger-анимации на 30+ элементах

### 10.4 Reduced motion

```css
@media (prefers-reduced-motion: reduce) {
  *,
  *::before,
  *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
}
```

В Framer Motion: использовать `useReducedMotion()`
и отдавать `false` варианты.

## 11. Декоративные элементы

### 11.1 Ботанические завитки (sparingly)

В качестве **тонких** декоративных штрихов:

- В фоне Hero (низкая прозрачность)
- В углах больших секций (CTA-блоки)
- В футере как разделитель

**Источник:** кастомные SVG в стиле гравюр
викторианских ботанических атласов (стилизация,
не реальные иллюстрации) или **noun project /
streamline botanical** с лицензией.

Цвет: `sepia-500` на `opacity: 0.15–0.25`.

### 11.2 Текстура «бумага» (опционально)

Мягкий шум на больших светлых поверх

### 11.2 Текстура «бумага» (опционально)

Мягкий шум на больших светлых поверхностях
(Hero, CTA-блоки):

```css
.paper-texture {
  background-image: url('/textures/paper-noise.png');
  background-blend-mode: multiply;
  opacity: 0.4;
}
```

- Размер тайла: 256×256px
- Вес файла: ≤ 30KB
- Подключать **только** к декоративным слоям,
  не к тексту
- Можно опустить на старте, добавить в v1.1

### 11.3 Разделители секций

Вместо классической `<hr>`:

- Тонкий SVG-завиток по центру (24px высотой,
  `sepia-300`)
- Или просто **воздух** (96–128px пустоты)

**Принцип:** воздух — главный разделитель.
Линии — крайний случай.

## 12. Состояния (states)

### 12.1 Hover

- Карточки: `translateY(-4px)` + усиление тени
- Кнопки: смена фона + лёгкий glow
- Ссылки в тексте: подчёркивание появляется снизу
  (через псевдоэлемент с `transform: scaleX`)
- Иконки-кнопки: фон становится `ink-glass`

### 12.2 Focus (клавиатура)

- Visible focus ring: `outline: 2px solid moss-500` +
  `outline-offset: 3px`
- На светлом стекле — ring `moss-700`
- На фото — ring `cream-50` с тёмной обводкой

**Никогда не убирать `outline` без замены.**

### 12.3 Active (нажатие)

- Кнопки: `translateY(1px)` + слабее тень
- Длительность: 80ms

### 12.4 Disabled

- `opacity: 0.5`
- `cursor: not-allowed`
- Без hover/active эффектов

### 12.5 Loading

- Скелетоны: пульсирующий `cream-100 ↔ cream-200`
- Анимация: 1.5s ease-in-out infinite
- На кнопках: spinner (Lucide `Loader2` + rotation)

### 12.6 Empty

Пустой каталог / нет результатов:

- Иконка (Lucide, 48px, `sepia-300`)
- Заголовок: «Здесь пока пусто»
- Подпись: 1 строка с подсказкой
- CTA: «Заказать под заказ» / «На главную»

### 12.7 Error

Ошибка формы / API:

- Подсветка поля: бордер `state-error`
- Сообщение: красный текст под полем (sm, 500)
- Тост: красный фон, белый текст, 5 сек

## 13. Микро-взаимодействия (опционально, со вкусом)

### 13.1 Карточка товара

- На hover — фото `scale(1.05)` за 400ms
- Бейдж статуса — без изменений
- Подпись цены — лёгкий цветовой shift к `moss-700`

### 13.2 Кнопка CTA

- На hover — стрелка `translateX(4px)` за 200ms
- На focus — glow-ring пульсирует один раз

### 13.3 Хедер

- При скролле > 50px — появляется `glass-light`
  фон + тонкий border-bottom
- Высота уменьшается с 80px → 64px (на десктопе)
- Transition: 300ms

### 13.4 Lightbox

- Открытие через Framer Motion `layoutId`:
  фото из карточки «вырастает» до полноразмера
- Закрытие — обратно к карточке

### 13.5 Модалка заказа

- Каналы появляются stagger'ом по 60ms
  (Telegram → WhatsApp → Звонок → Форма)
- На hover канала — иконка `scale(1.1)`

## 14. Адаптивные правила

### 14.1 Мобильный (320–767px)

- Контейнер: padding 16px по бокам
- Сетка карточек: 1 колонка
- H1: размер 4xl–5xl
- Hero: высота auto, не 100vh
- Меню: drawer справа (≥80% ширины)
- Touch targets: ≥ 44×44px
- Шрифт основной: размер base без уменьшения

### 14.2 Планшет (768–1023px)

- Контейнер: padding 32px
- Сетка карточек: 2 колонки
- Hero: высота 80vh
- Меню: горизонтальное, без drawer
- Появляются hover-эффекты (но не критично —
  тач всё ещё в приоритете)

### 14.3 Десктоп (1024–1279px)

- Контейнер: padding 40px
- Сетка карточек: 3 колонки
- Hero: 100vh с ограничением min-height 720px
- Floating CTA внизу-справа

### 14.4 Wide (1280px+)

- Контейнер: max-width 1280px по центру
- Padding боковой: вычисляется автоматически
- Без увеличения сетки карточек (остаётся 3)
- Можно увеличить отступы между секциями
  (до space-32)

## 15. Темы / Variants (на будущее)

На старте — **только светлая тема**.

Но токены организованы так, что добавление тёмной
темы — это новый файл с переопределением переменных.
Структура готова к расширению.

В CSS:

```css
:root {
  /* light theme */
}
[data-theme='dark'] {
  /* однажды, может быть */
}
```

## 16. Брендинг

### 16.1 Логотип

**Текстовый**, без иконки:

- Шрифт: Fraunces 500, размер xl–2xl
- Текст: «Флорки»
- Опциональный микро-засекр: завиток в букве «Ф»
  (через `axes: { WONK: 1 }`)
- Цвет: `sepia-900`
- На тёмных подложках: `cream-50`

### 16.2 Tagline (под логотипом, опционально)

- Шрифт: Inter 500, размер xs uppercase
- Letter-spacing: 0.1em
- Текст: «Флорариумы • Ярославль»
- Цвет: `sepia-500`

### 16.3 Favicon

- 32×32 + 16×16 ICO
- Apple touch icon: 180×180 PNG
- SVG-фавикон: «Ф» в Fraunces
- Цвет фона: `cream-50`
- Цвет буквы: `moss-500`

## 17. Tone of voice (в текстах)

Не часть визуала, но влияет на восприятие.

### 17.1 Голос бренда

- **Тёплый, спокойный, уверенный**
- Без «эксклюзивных предложений» и «уникальных
  возможностей»
- Без обращения «Вы» с заглавной (это не деловая
  переписка)
- Можно «вы», можно «ты» в зависимости от контекста,
  но **последовательно** в рамках одной страницы

### 17.2 Примеры

❌ «Уникальные авторские флорариумы по выгодным
ценам! Закажите прямо сейчас!»

✅ «Делаем флорариумы в Ярославле. Каждый —
маленький мир под стеклом.»

❌ «Оставьте заявку и наш менеджер свяжется с Вами
в ближайшее время.»

✅ «Напишите в Telegram — ответим за 15 минут.»

### 17.3 Стоп-слова

- «Уникальный», «эксклюзивный», «премиальный»
- «Команда профессионалов»
- «Индивидуальный подход»
- «Качественный сервис»
- «Дорогие клиенты»

## 18. Чек-лист визуальной готовности

Перед запуском визуально:

### Палитра

- [ ] Все цвета — из палитры, ни одного «случайного» hex
- [ ] Контрасты прошли AA
- [ ] Sepia-300 нигде не используется как текст

### Типографика

- [ ] Заголовки только в Fraunces
- [ ] Основной текст только в Inter
- [ ] Длинное тире (—), кавычки-ёлочки
- [ ] Висячие союзы заменены на &nbsp;
- [ ] Цены с табулярными цифрами

### Композиция

- [ ] Отступы кратны 4px / 8px
- [ ] Радиусы из шкалы (sm/md/lg/xl/2xl/full)
- [ ] Тени только из шкалы (тёплые)
- [ ] Сетка контейнера соблюдена

### Стекло

- [ ] Применено только там, где уместно
- [ ] Fallback для старых браузеров есть
- [ ] Не используется на длинном тексте

### Анимация

- [ ] Все transitions ≤ 300ms (UI)
- [ ] Reduced motion обработан
- [ ] Нет parallax, custom cursor, autoplay

### Фото

- [ ] Только реальные работы
- [ ] WebP, оптимизированные
- [ ] Alt-тексты осмысленные
- [ ] Соотношения сторон выдержаны

### Иконки

- [ ] Только Lucide
- [ ] Stroke-width 1.75
- [ ] Размеры из шкалы (20/24/32)

### Брендинг

- [ ] Логотип единый на всех страницах
- [ ] Favicon заменён со стандартного Next.js
- [ ] OG-картинки бренд-айдентичные

### Tone of voice

- [ ] Нет стоп-слов
- [ ] Тексты живые, не корпоративные
- [ ] Везде последовательное обращение
