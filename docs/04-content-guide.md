## 16. Ответственный за контент

### 16.1 Роли

- **Автор / редактор** — основательница студии
  (Анастасия) или копирайтер по согласованию
- **Финальная проверка** — арт-директор (визуально
  тексты ложатся в макет)
- **Корректура** — внешний редактор перед запуском
  (опционально, но желательно)

### 16.2 Цикл правок

1. Черновик в Google Docs / Notion
2. Проверка по чек-листу (раздел 13)
3. Согласование с арт-директором
4. Перенос в MDX / на сайт
5. Финальный просмотр на превью (как ложится)
6. Публикация

### 16.3 Когда нужна срочная правка

- Опечатка / ошибка → исправляем сразу, без цикла
- Изменение цены → исправляем сразу
- Новый товар → стандартный цикл (1–2 дня)
- Новая статья → 1–2 недели (написать +
  оформить + согласовать)

## 18. Где хранится контент

### 18.1 Тексты страниц

- Главная, О нас, Контакты → в коде компонентов
  (на старте)
- Политика, Оферта → MDX-файлы в `/content/legal/`
- Статьи → MDX в `/content/articles/`
- Карточки товаров → JSON в `/data/products.json`
  (или MDX в `/content/products/`)
- Отзывы → JSON в `/data/reviews.json`

### 18.2 Фото

- Все изображения → `/public/images/`
  с подпапками:
  - `/products/{slug}/` — фото товаров
  - `/articles/{slug}/` — фото статей
  - `/about/` — фото студии и команды
  - `/hero/` — обложки секций

### 18.3 Резервные копии

- Все исходники (полноразмерные JPG / RAW) —
  в Google Drive / Яндекс.Диске **отдельно от репо**
- В репо — только оптимизированные WebP / JPG
- Раз в месяц — бэкап репозитория (GitHub +
  локальная копия)

## 19. Лицензии и авторские права

### 19.1 Фотографии

- **Все фото — собственные**, сделанные студией
  или с разрешения клиента
- На фото клиентских заказов — спросить, можно ли
  публиковать (особенно если в кадре их интерьер)
- **Никаких стоковых фото** в карточках товаров
  и галерее
- Стоковые фото допустимы **только** как
  иллюстрации в статьях (с указанием источника),
  но лучше избегать

### 19.2 Тексты

- Все тексты — оригинальные
- Цитаты из других источников — с указанием
  автора и ссылкой
- Не копируем описания товаров с конкурентов
- Не используем нейросеть для написания финальных
  текстов (черновик — можно, но переписать
  своими словами обязательно)

### 19.3 Иконки и шрифты

- **Lucide Icons** — лицензия ISC, бесплатно
  для коммерческого использования
- **Fraunces, Inter** — Google Fonts, OFL/SIL,
  бесплатно
- Декоративные SVG — собственные или с
  открытыми лицензиями (CC0, MIT)

## 20. Контент и SEO — баланс

### 20.1 Принцип

**Сначала пишем для людей, потом оптимизируем
для поиска.** Не наоборот.

Если текст звучит как SEO-простыня — он не
работает ни для людей, ни для Яндекса (новые
алгоритмы это понимают).

### 20.2 Что точно делаем

- Уникальный title и description на каждой
  странице
- H1 — один на странице, содержит ключевую фразу
  естественно
- Alt у всех картинок
- Ссылки между связанными страницами
  (карточка товара ↔ статья об уходе)
- Чистые URL (`/katalog/mshistyi-les`,
  не `?id=42`)

### 20.3 Что НЕ делаем

- ❌ Не пихаем «флорариум Ярославль купить»
  в каждый абзац
- ❌ Не пишем «портянки» по 5000 слов «для SEO»
- ❌ Не используем серую оптимизацию (cloaking,
  скрытый текст)
- ❌ Не покупаем ссылки
- ❌ Не размещаем тексты на сомнительных биржах

## 21. Финальный чек-лист контентной готовности

Перед запуском сайта проверить:

### Главная

- [ ] H1 без пафоса и стоп-слов
- [ ] USP-блок: 4 пункта с конкретикой
- [ ] CTA внизу страницы
- [ ] Хотя бы 4–6 карточек товаров в featured

### Каталог

- [ ] Все товары имеют название, фото, цену,
      габариты, состав
- [ ] У каждого товара — статус (в наличии /
      под заказ)
- [ ] Alt-тексты осмысленные

### Карточка товара

- [ ] Описание 2–4 абзаца, без воды
- [ ] Уход — 1–2 предложения + ссылка на статью
- [ ] Цена без копеек
- [ ] Кнопка «Написать» работает (Telegram /
      WhatsApp / форма)

### Отзывы

- [ ] Минимум 8 отзывов
- [ ] Все — дословные, с указанием площадки
- [ ] Есть ссылки на оригиналы

### Полезное

- [ ] Минимум 3 статьи
- [ ] У каждой статьи — обложка, описание,
      теги, дата
- [ ] Время чтения указано
- [ ] CTA в конце статьи

### О студии

- [ ] Текст 400–600 слов, личный, без корпоратива
- [ ] Фото основательницы / студии
- [ ] Что мы делаем, как работаем, ценности

### Контакты

- [ ] Адрес (или район + по записи)
- [ ] Часы работы конкретные
- [ ] Telegram, WhatsApp, телефон, email
- [ ] Карта (если есть конкретный адрес)
- [ ] Форма работает

### Правовое

- [ ] Политика конфиденциальности
- [ ] Оферта
- [ ] Реквизиты (ИП / самозанятый)

### SEO

- [ ] Title и description на каждой странице
- [ ] Open Graph картинка
- [ ] Favicon
- [ ] robots.txt
- [ ] sitemap.xml

### Общее

- [ ] Нет Lorem ipsum
- [ ] Нет «TODO» в текстах
- [ ] Нет тестовых телефонов вида +7 (000) 000-00-00
- [ ] Все ссылки работают
- [ ] Все картинки загружаются
