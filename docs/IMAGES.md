# Требования к изображениям — florki.ru

Все изображения загружаются через `next/image` (WebP + AVIF автоконверсия).  
Исходники сдавать в **JPG или PNG, минимум 2000px по длинной стороне**.  
Файлы класть точно по указанным путям — код ссылается на них напрямую.

## 1. Hero — главный экран

### `/public/images/hero/florarium.png`

| Параметр     | Значение                   |
| ------------ | -------------------------- |
| Формат       | **PNG с прозрачным фоном** |
| Размер файла | рекомендуемый 1000×1000 px |
| Приоритет    | 🔴 Критический             |

**Что снимать:**  
Один красивый флорариум — лучше всего закрытая стеклянная сфера или домик с мхами.  
Снимать на белом или однотонном фоне, потом вырезать фон (или снять на планшете с белым листом).  
Флорариум должен стоять прямо, занимать бо́льшую часть кадра.

**Зачем PNG:** изображение накладывается поверх градиентного фона hero-секции с эффектом свечения — белый фон убьёт эффект.

**Используется в:** `HeroSection.tsx` — правая колонка главного экрана.

## Технические требования

### Формат и качество

- Сдавать исходники **JPG/PNG ≥ 2000px** по длинной стороне
- Next.js автоматически конвертирует в WebP/AVIF при первом запросе
- **Исключение:** `/images/hero/florarium.png` и `/images/og/default.png` — только PNG

### Оптимизация исходников

- JPG: качество 85–90%, без лишних метаданных
- Размер файла рекомендуется < 2 МБ на исходник
- Цветовой профиль: sRGB
