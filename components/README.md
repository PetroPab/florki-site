# components/

Переиспользуемые React-компоненты. Разделены по назначению.

## Структура

```
components/
├── ui/        ← базовые примитивы
├── layout/    ← шапка, подвал, обёртки
├── sections/  ← полноширинные секции страниц
├── blocks/    ← карточки и блоки контента
├── forms/     ← формы
├── modals/    ← модальные окна
└── seo/       ← JSON-LD разметка
```

---

## `ui/` — базовые примитивы

| Компонент           | Описание                                                                                  |
| ------------------- | ----------------------------------------------------------------------------------------- |
| `Button`            | Кнопка с вариантами `primary`, `secondary`, `ghost`; поддержка `btn-shimmer`, `btn-pulse` |
| `Badge`             | Бейдж/тег для категорий и статусов                                                        |
| `Card`              | Карточка-контейнер с glassmorphism-вариантом                                              |
| `Container`         | Центрирующая обёртка с отступами (`max-w-7xl`)                                            |
| `Heading`           | Типографика заголовков с поддержкой `display-em`                                          |
| `Text`              | Параграф с вариантами размера и цвета                                                     |
| `Input`             | Текстовое поле формы                                                                      |
| `Textarea`          | Многострочное поле формы                                                                  |
| `Checkbox`          | Чекбокс с кастомным стилем                                                                |
| `Link`              | Ссылка (обёртка над `next/link`)                                                          |
| `ImageWithFallback` | `next/image` с placeholder при ошибке загрузки                                            |

---

## `layout/` — структура страницы

| Компонент    | Описание                                                                         |
| ------------ | -------------------------------------------------------------------------------- |
| `Header`     | Навигация сайта: логотип, меню, кнопка заказа. Мобильное меню через `MobileMenu` |
| `Footer`     | Подвал: контакты, навигация, юридические данные                                  |
| `MobileMenu` | Выдвижное меню для мобильных (Radix Dialog)                                      |
| `Section`    | Секция страницы: `className`, `id`, padding, фон                                 |

---

## `sections/` — секции главной страницы

| Компонент          | Описание                                            |
| ------------------ | --------------------------------------------------- |
| `HeroSection`      | Первый экран: заголовок, подзаголовок, CTA-кнопки   |
| `FeaturedProducts` | Блок избранных товаров (`featured: true`)           |
| `USPSection`       | Преимущества студии (3–4 карточки)                  |
| `ReviewsStrip`     | Горизонтальная лента отзывов                        |
| `CTASection`       | Финальный призыв к действию + форма быстрого заказа |

---

## `blocks/` — карточки и блоки контента

| Компонент      | Описание                                          |
| -------------- | ------------------------------------------------- |
| `ProductCard`  | Карточка товара для каталога                      |
| `ArticleCard`  | Карточка статьи для /poleznoe                     |
| `ReviewCard`   | Карточка отзыва                                   |
| `GalleryGrid`  | Сетка галереи с masonry-раскладкой                |
| `AnimatedGrid` | Grid с анимацией появления (scroll-reveal)        |
| `Lightbox`     | Просмотр фото в полный экран                      |
| `GlassPanel`   | Glassmorphism-панель для произвольного контента   |
| `Callout`      | Выделенный блок в статьях (совет, предупреждение) |
| `OrderButton`  | Кнопка «Заказать» — открывает `OrderModal`        |

---

## `forms/`

| Компонент     | Описание                                                                            |
| ------------- | ----------------------------------------------------------------------------------- |
| `ContactForm` | Форма обратной связи (React Hook Form + Zod). Отправляет на `/api/contact` → Resend |

---

## `modals/`

| Компонент    | Описание                                                                                   |
| ------------ | ------------------------------------------------------------------------------------------ |
| `OrderModal` | Модалка выбора канала связи: Telegram / WhatsApp / Звонок / Форма. Принимает `productName` |

---

## `seo/`

| Компонент             | Описание                                              |
| --------------------- | ----------------------------------------------------- |
| `JsonLdLocalBusiness` | JSON-LD разметка `LocalBusiness` для главной страницы |
| `JsonLdProduct`       | JSON-LD разметка `Product` для страниц товаров        |
